# Bee-Tracking

## 실시간 꿀벌 모니터링

## 프로젝트 개요
비 벡터링의 효과를 극대화하기 위해 꿀벌을 추적(Tracking)하여, 소비자가 웹 페이지를 통해 꿀벌의 상태를 실시간으로 확인할 수 있는 서비스를 개발했습니다.
